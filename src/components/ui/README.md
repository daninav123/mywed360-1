# UI Kit (base)

Componentes reutilizables recomendados:
- Button, IconButton
- Input, Select, Textarea, Checkbox, Switch
- Modal, Drawer
- Table (con paginación y estados vacíos)
- EmptyState (título, descripción, CTA)
- Loader (spinners)

Estándares:
- Tamaños: `sm`, `md`, `lg`.
- Variantes: `primary`, `secondary`, `ghost`, `danger`.
- Estados: `disabled`, `loading`, `aria-*` y focus visible.

Usar Tailwind para layout y estados; MUI cuando aporte accesibilidad avanzada.
