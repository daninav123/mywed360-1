# Requisitos Legales por País

Esta carpeta contiene la información verificada de requisitos legales para matrimonio en diferentes países.

## Estructura

Cada país tiene su propia carpeta con archivos específicos:

```
requisitos-legales/
├── README.md (este archivo)
├── europa/
│   ├── ES-espana.md
│   ├── FR-francia.md
│   ├── IT-italia.md
│   └── ...
├── america/
│   ├── US-estados-unidos.md
│   ├── MX-mexico.md
│   ├── CA-canada.md
│   └── ...
└── PLANTILLA.md (plantilla para nuevos países)
```

## Instrucciones para rellenar

1. Copia `PLANTILLA.md` y renómbrala según el país (ej: `ES-espana.md`)
2. Rellena todos los campos con información verificada de fuentes oficiales
3. Incluye siempre enlaces a las fuentes oficiales
4. Fecha de última actualización
5. Notas sobre variaciones regionales si aplica

## Nivel de Verificación

Marca cada documento con su nivel de verificación:
- ✅ **VERIFICADO**: Información confirmada con fuentes oficiales actuales
- ⚠️ **PARCIAL**: Información general, pendiente de verificación detallada
- ❌ **NO VERIFICADO**: Información aproximada, requiere verificación urgente

## Actualización

Esta información debe revisarse cada **6 meses** mínimo, ya que las leyes pueden cambiar.
